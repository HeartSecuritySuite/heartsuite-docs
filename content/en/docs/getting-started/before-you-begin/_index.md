Before installing HeartSuite, ensure your system meets these requirements to avoid issues during setup.

## System Requirements

- **Operating System**: A Debian 11 or Ubuntu-compatible Linux server on x86 architecture.
- **Access Level**: Root access (sudo privileges).
- **Skills**: Basic familiarity with the command line (we'll guide you through steps).

If your setup differs, check the [Introduction](../introduction/) for compatibility details.
